1. Intro JS
    1) What is JS
        - Scripting Language scripting means interpreted​
        - Used both on the side​
        - Client Side​
        - Server Side​
        - Event Based Language​
        - Object Based Language – it is based on object oriented concepts.​
        - Light weight Language – it does not takes a lot memory to run the code.​
        - Cross platform Language- it can be run on different platforms.​


    2) Why JS
        - Help us to create dynamically updating content.​
        - Converts the static webpage into a interactive web page.​
        - For security and Validation.​
        - To fetch the information like IP Address, Browser Details etc.​
        - Can load content without reloading the entire page.​
        - Games Development.​


    3) JS Use
        - Web Client Side: Js, Jquery, React JS​
        - Web Server Side: NodeJS, Express JS​
        - Mobile Development: React Native, PhoneGap, Ionic​
        - S/w Development: electroJS, Example – VSCode​
    
    
    4) How to embedded the JS
        i) Internal Link​
            <script>​
            …​
            </script>​

        ii) External Link​
            <script src=“…”>​
        
        - We can use the script tag in <head> or <body> tags.
        - If you want to load JS Code before <html> then use it in <head> tag
        - But if you want to load JS Code after <html> then use <body> tag


    5) JS limitations
        - JS is a Backwards compatible. It means the Old Js code will work on the Latest Browsers.
        - JS is not a forward compatible. It means Latest code of JS will not work on the old browser.

                            Added async functions, Added shared memory
